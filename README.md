# C#, .NET 8 - Microservice

## Pre-requirements

- .NET 8 SDK
- Visual Studio Code with:
    - C# Extension
    - (optional) REST Client Extension
- (optional) Docker Desktop WSL 2 for Windows
- (optional) Postman

## Technologies
- C# .NET 8
- ASP.NET Web API (Minimal APIs)
- Dependency Injection
- XUnit, Moq
- Swagger
- Docker & docker-compose
